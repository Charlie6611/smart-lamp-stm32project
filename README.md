# smart-lamp-stm32project
基于stm32f103c8t6的智能台灯
使用esp8266连接机智云网络，实时查看参数，手机控制。
手动模式下可以自行调节led亮度，传感器阈值
自动模式下由光敏电阻调节led亮度
内部设有rtc实时时钟
